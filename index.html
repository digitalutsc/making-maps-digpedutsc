Why Make Maps?


  • Visualize complex data
  • Track information over distance and time
  • Teach students a new way to engage with data and digital scholarship
  • Give students a chance interact with primary source material directly

MapsEngine


  • Google product - works with any Google ID (no extra credentials required)
  • Make layers, change colours/shape of pins, add images, directions, etc.
  • Easy and intuitive


mapsengine.google.com

Export Data and Manipulate!


  • From MapsEngine, export data as KML
    • Keyhole Markup Language - for geographical data
  • Convert to geoJSON using ogr2ogr
  • Import into complex mapping software, like Leaflet.js

Leaflet.js


  • Open source, maintained by community
  • Uses Javascript
  • Mobile friendly
  • Highly customizable: community built plugins and code your own additional features


leafletjs.com
